# Tascherl - Pitch Demo

## 🎯 iOS 18 Liquid Glass Design - iPhone 17

A beautiful interactive pitch demo for Tascherl, the digital wallet app.

### 🚀 Features
- ✨ iOS 18 Liquid Glass Effect (glassmorphism)
- 📱 iPhone 17 Pro Max Frame with Dynamic Island
- 💳 Card Management (Barcode, Member Cards, NFC)
- 📍 Location-based Recommendations
- 🔒 Security Features Showcase
- 🎨 Golden Circle Visualization
- 🎬 Smooth Framer Motion Animations

### 📦 Tech Stack
- **React 18** - UI Framework
- **Vite** - Lightning-fast build tool
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Advanced animations
- **Lucide React** - Icon library

### 🛠️ Setup

```bash
npm install
npm run dev
```

The demo will open at `http://localhost:5173`

### 📸 Screens
1. **Home** - Card overview & quick actions
2. **My Cards** - Card details with QR code
3. **Add Card** - Multi-type card creation
4. **Nearby** - Location-based offers
5. **Security** - Data protection features
6. **Golden Circle** - Why/How/What visualization

### 🎨 Design Details
- **Color Scheme**: Dark OLED-optimized blacks
- **Typography**: System SF Pro Display font
- **Effects**: Dual blur layers for depth
- **Interactions**: Smooth tap & scroll animations

### 📱 Responsive
Designed to look like a real iPhone on any screen size.
